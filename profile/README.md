## 团队使命 🎯
我们的目标是通过开发和运营各种开源项目，为开发者提供更多的便利与创新工具，共同推动开源社区的发展。

---

## 核心项目 🚀
### 绮宫梦莉
高性能的 QQ/Kook 机器人！功能丰富，性能卓越。

### PAPM
高仿 npm 的 PHP 包管理器，为 PHP 开发者提供极致的依赖管理体验。

### Koishi 插件
功能简单而独特的插件，为 Koishi 平台带来更多可能性。
